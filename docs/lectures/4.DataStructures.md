## ğŸŸ¢ 2. Tuples

**Tuples:**
- Ordered, immutable collection.

- Faster than lists.

- Suitable for fixed data.



```python
# Creating a tuple

print(dimensions)

# Indexing and Slicing
print(dimensions[1])
print(dimensions[:2])

# Looping
for dim in dimensions:
    print(dim)

```

**â¤ List vs Tuple:**

- List: **Mutable** â†’ You can modify contents.

- Tuple: **Immutable** â†’ Contents cannot be changed.

- Lists have more built-in methods compared to tuples.

- Use lists for homogeneous items and tuples for heterogeneous items.




**â¤ Switching Between Lists and Tuples:**


```python

my_tuple = tuple(my_list)
print(my_tuple)

# Tuple to List

print(new_list)
```

**â¤ Tuple Unpacking:**

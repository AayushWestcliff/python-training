## ðŸŸ¢ 2. Generators & Iterators

**Generators:**
- Use `yield` to produce a sequence of values lazily (one at a time).

- Saves memory compared to lists.

**Iterators:**
- Any object with `__iter__()` and `__next__()` methods.


**âž¤ Generator Example:**


```python
def count_up_to(n):
    count = 1
    while count <= n:
        yield count
        count += 1
print(count_up_to(5))  # This will print a generator object
print(list(count_up_to(5)))
for num in count_up_to(5):
    print(num)
```

**âž¤ Iterator Example:**


```python
nums = [1, 2, 3]
it = iter(nums)

print(next(it))
print(next(it))
print(next(it))
```

## ðŸŸ¢ 3. List Comprehensions

```A concise way to create lists in a single line.```

**âž¤ Example:**


```python
# Without list comprehension
squares = []
for i in range(5):
    squares.append(i ** 2)
print(squares)

# With list comprehension
squares = [i ** 2 for i in range(5)]
print(squares)

# With condition
evens = [i for i in range(10) if i % 2 == 0]
print(evens)

```

## ðŸŸ¢ 4. Working with *args and **kwargs

- `*args`: Allows variable number of positional arguments.

- `**kwargs`: Allows variable number of keyword arguments.

**âž¤ Example with \*args:**


```python
def add(*numbers):
    total = sum(numbers)
    print("Sum:", total)

add(1, 2, 3)
add(4, 5)
```

**âž¤ Example with \*\*kwargs:**


```python
def student_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

student_info(name="Alice", age=21, grade="A")
```

**âž¤ Using map() Function:**


```python

squared = list(map(lambda x: x ** 2, nums))
print(squared)
```

**âž¤ Using filter() Function:**


```python
nums = [1, 2, 3, 4, 5, 6]
evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)
```

**âž¤ Using reduce() Function**
- `reduce()` applies a function cumulatively to the items of a sequence.

- It is part of the `functools` module.


```python
from functools import reduce

# Multiply all numbers in a list
nums = [1, 2, 3, 4]

product = reduce(lambda x, y: x * y, nums)
print(product)

```


```python
nums = [5, 10, 15]

total_sum = reduce(lambda x, y: x + y, nums)
print(total_sum)
```

**âž¤ Using enumerate():**


```python

    print(index, fruit)
```
